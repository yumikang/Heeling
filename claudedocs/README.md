# claudedocs/

Claude 및 AI 어시스턴트가 코드 작성 시 참조하는 기술 문서입니다.

## 문서 목록

| 파일 | 설명 |
|------|------|
| **BACKEND_API.md** | 백엔드 API 엔드포인트 스펙 |
| **mobile-api-spec.md** | 모바일 앱 API 요청/응답 스펙 |
| **api-usage-guide.md** | API 클라이언트 사용법 및 예제 |
| **sync-service-guide.md** | SyncService 동기화 로직 가이드 |

## docs/ 폴더와의 차이

- **docs/** = 사람이 읽는 문서 (가이드, 절차서, 요구사항)
- **claudedocs/** = 기술 참조 문서 (API 스펙, 코드 패턴, 타입 정의)

## 관리 원칙

1. API 스펙 변경 시 이 폴더 문서도 함께 업데이트
2. 코드와 문서의 일관성 유지
3. 완료된 임시 작업 문서는 삭제
